Technologies Used:

Python, HTML, CSS, Twitter Bootstrap, Flask

I came up with the idea for a suite of very simple apps to help with a few mundane tasks at work. Much of my daily work duties involve repetitive tasks where only a few variables change within a larger set structure - a suitable problem for programming to solve.

Most the applications are simple web forms that insert variables into email templates, but there are some more complicated tools such as a date checker and a SQL database-driven notice board. For as simple as these programs are, they are without a doubt the most personally useful code I've yet produced.

The apps save me so much time at work that once I let my boss know about the project she had me release it to our department and we've been using CSTools ever since.
