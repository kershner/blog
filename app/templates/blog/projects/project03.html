Technologies Used:

Python, HTML, CSS, JavaScript, Flask, Linux, Decoupage!

This project was great fun because it combined my love of hardware with my new found interest in programming. The Raspberry Pi allows you to see your code manifest in the real world on an actual physical object, and that is really powerful. This project isn't really as practical as a lot of Pi projects, but it is endlessly entertaining. It also turned out to have a much wider scope than I initially thought, forcing me to step outside my comfort zone often and with great results.

The code for the project is linked above with my Github along with a separate configuration website I created for the device.

The project is pretty much finished from both a software and artistic perspective. Next up, when I find the time, I'll be completely rewriting my project writeup as what I currently have up is woefully outdated and incomplete.
